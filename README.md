# io_messenger

IOTA based messenger

## With the Ionic CLI:

```bash
$ sudo npm install -g ionic cordova
```

Then, to run it, cd into `io_messenger` and run:

```bash
$ ionic cordova platform add android
$ ionic cordova run android
```
